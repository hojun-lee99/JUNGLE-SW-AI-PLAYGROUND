## 1번 문제
1. 파이썬에서 문자열을 처리하는 방법에는 문자열(str)을 그대로 사용하는 방법과, 문자열을 문자들의 리스트(list)로 변환하여 사용하는 방법이 있다.

예를 들어 다음과 같은 두 코드가 있다.

```python
s = "hello"
# 방법 1
result1 = s + "!"

# 방법 2
lst = list(s)

lst.append("!")

result2 = "".join(lst)
```
문자열을 그대로 사용하는 방법과 문자열을 리스트로 변환하여 사용하는 방법은 각각 장단점이 있다.

문자열(str)의 특성과 리스트(list)의 특성을 고려하여 두 방법의 장점과 단점을 설명하시오. (2점)

## 3번 문제
3. 다음 코드의 출력 결과를 예측하고, 왜 그렇게 되는지 각각 설명하시오.

```python
def modify(a, b, c):
    a += 1
    b.append(4)
    c = [100, 200]

x = 10
y = [1, 2, 3]
z = [7, 8, 9]

modify(x, y, z)

print(x)  # (1)
print(y)  # (2)
print(z)  # (3)
```

```python
# 실행결과
10
[1, 2, 3, 4]
[7, 8, 9]
```
이유
modify 함수에서,  
x는 modify 함수 내의 지역변수 a의 값을 업데이트 해주었다 생각했기 때문에 10을 출력했고
y는 modify 함수 내에서 append()라는 리스트 클래스의 메서드를 사용해 메모리에 저장된 y의 주소를 참조해 데이터를 추가해주었다고 생각하기 때문에 4가 마지막에 추가관 [1, 2, 3, 4]를 출력했고
z는 modify 함수 내에서 c = [100, 200] 문장은 z 리스트를 업데이트 해준 것이 아닌 100, 200이 담긴 새로운 리스트 지역 변수를 선언했다고 생각했기 때문에 출력은 [7, 8, 9]로 출력될 것으로 예상했습니다.

## 6번 문제
6. 한 절에는 여러 명의 스님이 살고 있다. 
  병에 걸린 스님은 머리에 붉은 점이 생기며, 스님들은 매일 아침 한자리에 모여 서로의 얼굴을 볼 수 있지만 자신의 얼굴은 볼 수 없다. 
  어느 날 아침, 주지 스님이 모든 스님에게 “이 절에는 머리에 붉은 점이 있는 스님이 최소 한 명 이상 있다”고 말했다. 
  이 날을 0일차라고 하며, 그 이후로 병에 걸린 스님의 수는 변하지 않는다. 
  스님들은 매일 아침 서로의 머리를 확인하면서 자신이 병에 걸렸는지 논리적으로 판단한다. 
  어떤 스님이 자신의 머리에 붉은 점이 있다는 사실을 확실히 알게 되면, 그 스님은 다음 날 아침 모임에 나오지 않고 절을 떠난다. 
  그런데 1일차부터 6일차 아침까지는 아무도 절을 떠나지 않았다. 
  그러나 7일차 아침이 되자 머리에 붉은 점이 있던 스님들이 모두 동시에 절을 떠나 더 이상 보이지 않았다. 
  이때 머리에 붉은 점이 있던 스님은 몇 명이었는가? 또한 붉은 점이 있는 스님이 n명일 때, 모든 스님이 절을 떠나는 날을 반환하는 재귀 함수 departure_day(n)을 작성하라. (단, 주지 스님은 병에 걸리지 않았다.) (2점)
