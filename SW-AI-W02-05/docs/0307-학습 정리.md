## 파이썬 기본 문법

파이썬에서 컬렉션을 저장하는데 사용되는 4가지 내장 데이터 타입
- List 리스트
  - 순서가 있고, 변경 가능한 집합, 중복 멤버를 허용
- Tuple 튜플
  - 순서가 있고, 변경 불가능한 집합, 중복 멤버 허용
- Set 셋
  - 순서가 없고, 변경 불가능, 중복 멤버 금지, 인덱싱 되지 않음
- Dictionary 딕셔너리
  - 순서가 있고, 변경 가능한 모음, 중복 멤버 금지
  - *파이썬 3.7부터 순서가 있고 이전 버전은 순서가 없음

### List 리스트

하나의 변수에 여러 항목을 저장하는 데 사용함
- 다른 언어의 배열과 비슷해보임

```python
  thislist = ["apple", "banana", "cherry"]
  print(thislist)
  #출력결과
  # ['apple', 'banana', 'cherry']
```

순서를 가지고 있고 새 항목은 마지막에 추가 된다.
리스트의 항목 갯수를 알기위해선 `len()`함수를 사용
어떤 데이터든 담을수 있고 데이터의 타입이 달라도 상관없음

리스트 컬렉션의 타입은 list 클래스의 객체이다
클래스이기 때문에, `list()` 생성자를 이용해 리스트를 만들수도 있다.
```python
  thislist = list(("apple", "banana", "cherry"))
  print(thislist)
  # 실행결과
  # ['apple', 'banana', 'cherry']
  print(type(thislist))
  # 실행결과
  # <class 'list'>
```

인데스는 `0`부터 시작한다

```python
mylist = ['apple', 'banana', 'cherry', 'orange', 'kiwi']
print(mylist[1:4])
# 실행결과
# ['banana', 'cherry', 'orange']
```
